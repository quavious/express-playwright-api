Fake Partners

쿠팡 파트너스 이메일, 비밀번호 만으로 쿠팡 파트너스 계정 정식 승인 없이 제휴 링크와 상품 정보를 한번에 받아보실 수 있습니다.

1. Token 발급

API 엔드포인트 : API_HOST/token

REST 메소드 : POST

양식 데이터 : 이메일, 비밀번호

        {
            "email": USER_EMAIL,
            "password": USER_PASSWORD,
        }

Content Type : application/json

이메일 비밀번호를 양식 데이터로 하여금 POST 요청을 보냅니다. 서버 내에서 헤드리스 브라우저가 돌아가므로 조금 느릴 수 있습니다.

정상적인 응답에는 status와 auth, cookie 데이터가 있습니다.

응답 값 
        {
            "status": true,
            "auth": AUTH_STRING,
            "cookie": COOKIE_STRING
        }

auth, cookie 모두 데이터를 요청하는데 필요합니다.


2. 데이터 요청

API 엔드포인트 : API_HOST/search

auth, cookie 문자열 데이터가 필요합니다. 검색어와 페이지 숫자를 JSON 요청 데이터에 넣어줍니다.

API_HOST/search에 POST 요청을 보냅니다.

요청 데이터 
        {
            "auth": AUTH_STRING,
            "cookie": COOKIE_STRING,
            "keyword": 검색어,
            "page": 페이지 숫자
        }

상품 정보와 상품과 본인의 쿠팡 파트너스 계정과 연동된 제휴 링크가 반환됩니다. 서버에서 광고 서버에게 연속적으로 AJAX 요청을 보내므로, 제휴 링크가 정상적으로 생성되지 않을 때가 있습니다.

status, response, message 데이터가 반환됩니다. response는 상품 정보, message는 제휴 링크 정상 생성 여부를 나타냅니다.
